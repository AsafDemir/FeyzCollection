# Feyz Collection — Görsel Dosyaları

Bu klasöre ürün fotoğraflarını aşağıdaki dosya adlarıyla kaydedin:

| Dosya adı              | Hangi fotoğraf?                              |
|------------------------|----------------------------------------------|
| `seccade-serili.jpg`   | Seccade yerde serili hali (geniş açı)       |
| `seccade-katli.jpg`    | Seccade katlı, nakış üstte görünür          |
| `nakis-detay-1.jpg`    | Nakış yakın plan (kubbe + kemerler)         |
| `nakis-detay-2.jpg`    | Nakış yakın plan (farklı açı / kırpım)      |
| `drawing.jpg`          | Kâğıt üzerindeki karakalem çizim            |

**Not:** JPG yerine PNG veya WebP de kullanabilirsiniz;
o zaman `index.html` içindeki `<img src="...">` yollarını güncellemeniz yeterli.
